Release for Minecraft 1.21.1.

- Adds the client-side `/particlemuffler inspect` command for short particle sampling.
- Supports default inspect arguments: 5 seconds, 32 results, sorted by count.
- Keeps Fabric and NeoForge published as separate Modrinth/CurseForge versions named by loader, Minecraft version, and SemVer.
- Uses payload-based networking for filtered muffler updates on Minecraft 1.21.1.
