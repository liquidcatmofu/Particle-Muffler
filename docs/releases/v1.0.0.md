Initial release for Minecraft 1.20.1.

- Adds Particle Muffler and Filtered Particle Muffler blocks.
- Supports Fabric and Forge.
- Suppresses particles client-side by section.
- Filtered muffler supports blacklist/whitelist particle ID rules with GUI editing.
- Redstone signal disables suppression.
- Includes TOML config for client registry pruning.
