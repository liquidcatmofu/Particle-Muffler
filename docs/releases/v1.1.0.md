Release for Minecraft 1.21.1.

- Replaces the Forge target with NeoForge.
- Updates menu, block entity, tooltip, resource location, and network code for 1.21.1 APIs.
- Registers NeoForge menu screens through the NeoForge mod event bus.
- Declares the particle suppression mixin for NeoForge.
